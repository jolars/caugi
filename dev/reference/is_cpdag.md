# Is the `caugi` graph a CPDAG?

Checks if the given `caugi` graph is a Complete Partially Directed
Acyclic Graph (CPDAG).

## Usage

``` r
is_cpdag(cg)
```

## Arguments

- cg:

  A `caugi` object.

## Value

A logical value indicating whether the graph is a CPDAG.

## Examples

``` r
cg_cpdag <- caugi(
  A %---% B,
  A %-->% C,
  B %-->% C,
  class = "PDAG"
)
is_cpdag(cg_cpdag) # TRUE
#> [1] FALSE

cg_not_cpdag <- caugi(
  A %---% B,
  A %---% C,
  B %-->% C,
  class = "PDAG"
)
is_cpdag(cg_not_cpdag) # FALSE
#> [1] FALSE
```
