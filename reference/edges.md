# Get edges of a `caugi`.

Get edges of a `caugi`.

## Usage

``` r
edges(cg)

E(cg)
```

## Arguments

- cg:

  A `caugi` object.

## Examples

``` r
cg <- caugi(
  A %-->% B,
  B %-->% C,
  D,
  class = "DAG"
)
edges(cg) # returns the tibble with columns from, edge, to
#> # A tibble: 2 Ã— 3
#>   from  edge  to   
#>   <chr> <chr> <chr>
#> 1 A     -->   B    
#> 2 B     -->   C
```
