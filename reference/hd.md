# Hamming Distance

Compute the Hamming Distance between two graphs.

## Usage

``` r
hd(cg1, cg2, normalized = FALSE)
```

## Arguments

- cg1:

  A `caugi` object.

- cg2:

  A `caugi` object.

- normalized:

  Logical; if `TRUE`, returns the normalized Hamming Distance.

## Value

An integer representing the Hamming Distance between the two graphs.
