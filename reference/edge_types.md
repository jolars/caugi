# Get the edge types of a `caugi`.

Get the edge types of a `caugi`.

## Usage

``` r
edge_types(cg)
```

## Arguments

- cg:

  A `caugi` object.

## Value

A character vector of edge types.

## Examples

``` r
cg <- caugi(
  A %-->% B,
  B %--o% C,
  C %<->% D,
  D %---% E,
  A %o-o% E,
  class = "UNKNOWN"
)
edge_types(cg) # returns c("-->", "o-o", "--o", "<->", "---")
#> [1] "-->" "o-o" "--o" "<->" "---"
```
