# Is it a `caugi` graph?

Checks if the given object is a `caugi`. Mostly used internally to
validate inputs.

## Usage

``` r
is_caugi(x, throw_error = FALSE)
```

## Arguments

- x:

  An object to check.

- throw_error:

  Logical; if `TRUE`, throws an error if `x` is not a `caugi`.

## Value

A logical value indicating whether the object is a `caugi`.
